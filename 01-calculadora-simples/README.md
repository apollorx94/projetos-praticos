# 🧮 Calculadora Simples

Calculadora de terminal que realiza as 4 operações básicas (soma, subtração, 
multiplicação e divisão), com tratamento de erros e formatação de resultado.
Implementada em PHP e JavaScript para fins de comparação e estudo.

## 🎯 Funcionalidades
- Operações básicas via funções dedicadas
- Tratamento de divisão por zero
- Validação de entrada (operação inválida)
- Formatação de resultado com 2 casas decimais
- Entrada interativa via terminal

## 🛠️ Tecnologias
- PHP 8.5
- Node.js + prompt-sync

## ▶️ Como executar

**PHP:**
\`\`\`bash
php php/calculadora.php
\`\`\`

**JavaScript:**
\`\`\`bash
cd javascript
npm install
node calculadora.js
\`\`\`

## 📚 O que aprendi neste projeto
- Funções com retorno (return) ao invés de impressão direta
- Validação de tipos (is_numeric / typeof)
- Entrada de dados via terminal
- Diferenças de sintaxe e comportamento entre PHP e JavaScript
